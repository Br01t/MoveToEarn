## ğŸ® Core Mechanics

### 1. Hexagonal World Map
The world is divided into a hexagonal grid. New zones are generated procedurally based on real-world GPS coordinates, clustering by country (e.g., IT, US, UK).

### 2. King of the Hill (Conquest)
Ownership isn't permanent. To keep a zone, you must defend it.
*   **Attack:** A challenger must run MORE kilometers in that specific zone than the current owner's record.
*   **Defense:** Owners can use **Shields** (24h protection) or simply run more to raise the bar.

### 3. Mission & Badge System
*   **100+ Unique Missions:** Ranging from "First 5k" to "Midnight Runner".
*   **Rarity Tiers:** Common, Rare, Epic, Legendary.
*   **Dynamic Progress:** Bars update in real-time based on your stats.

### 4. Anti-Cheat System
The `RunAnalysis` engine parses raw XML/GPX data to detect:
*   **Teleportation:** Impossible distance jumps.
*   **Speeding:** Avg speed > 25km/h or spikes > 50km/h (Vehicle detection).
*   **Duplicate Uploads:** Prevents re-uploading the same file ID/timestamp.
